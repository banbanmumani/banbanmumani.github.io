# 자주사용 하는 명령어
현제 경로 확인 하기 (Print Working Directory)
{% highlight bash %}
pwd
{% endhighlight %}

현제 경로 파일 목록 보기 (List)
{% highlight bash %}
ls
{% endhighlight %}

자주쓰는 옵션 -artl
* a '.' 으로 시작하는 (osx finder나 윈도우 탐색기에서 기본으로는 안보이게 설정) 파일도 보여주고
* r 역순정렬 하고
* t 최근 변경된 순으로 정렬하고
* l long 정보 표시
* 타입 권한 링크표시 소유자 소유자 용량 수정일 파일명
